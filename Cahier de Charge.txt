Roadmap : 

- Créer un compteur qui permet de changer l'affichage du site lorsqu'on clique sur les flèches.
- Définir les variables titre, flèches, liens Wikipédia, la description des personnes.
- Utiliser un liste de dictionnaire, avec comme clés : src (source de l'image), titre (nom du mec), description (description du mec ), wiki (lien de son wiki) 
- Lorsqu'on clique sur la flèche gauche, si le compteur est supérieur à 0, on lui retire 1. Sinon, on le remet à la dernière position de la liste (longueur du tableau - 1).
- Lorsqu'on clique sur la flèche droite, si le compteur est inférieur à la longueur de la liste - 1, on lui ajoute 1. Sinon, on le remet à 0 pour revenir au début de la liste.
- Optimiser le code au maximum afin d'éviter les if, else if.
- Avoir une page moderne avec un CSS pas très avancer
